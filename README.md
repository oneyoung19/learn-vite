- [ ] `vite` 能使用 `vue2` 么？
Vite 为 Vue 提供第一优先级支持：

Vue 3 单文件组件支持：@vitejs/plugin-vue
Vue 3 JSX 支持：@vitejs/plugin-vue-jsx
Vue 2.7 SFC 支持：@vitejs/plugin-vue2
Vue 2.7 JSX support via @vitejs/plugin-vue2-jsx
